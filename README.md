# bilibiliDownload
天朝大型同性交友网站(bilibili.com)的视频下载器

简单实现了：

视频下载

视频文件大小获取(单位MB)

用户输入格式检查

下载进度显示

## 使用
首先获取视频AV号：

例如：

http://www.bilibili.com/video/av23333/

该链接“av”后的纯数字即为视频AV号

输入后任意键确认，视频将自动下载到Videos目录（自动创建）
